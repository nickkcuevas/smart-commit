# Demo App

A simple demo application for Smart Commit tool demonstration.

## Features

- User management API
- RESTful endpoints
- Basic authentication
- Health check endpoint

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

## API Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check
- `GET /api/users` - List users
- `POST /api/users` - Create user
- `GET /api/users/<id>` - Get user by ID
